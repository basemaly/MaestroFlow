# Self-Critique Refinement

Use this skill when a single draft is not enough and the user wants visible improvement through iteration.

## Loop

1. Produce the best first draft you can.
2. Critique that draft against explicit criteria from the user or the task.
3. Revise the draft to fix the most important issues.
4. Keep the critique concise and focused on changes that materially improve the result.

## Good Criteria

- clarity
- usefulness
- specificity
- persuasiveness
- tone match
- structural coherence

## Rules

- Do not make the critique performative.
- Prefer 3 to 5 concrete observations over a long evaluation.
- The revised draft must be meaningfully better than the first one.
