# Humanize Writing

Use this skill when the user needs more than cosmetic cleanup. This is the deeper rewrite workflow.

## Priorities

- Make the voice sound intentional rather than statistically average
- Improve rhythm with sentence-length variety and cleaner paragraph flow
- Replace vague framing with concrete nouns, verbs, and examples
- Keep the original factual meaning unless the user asks for a stronger transformation

## Rewrite Process

1. Identify the target tone from the user's request and the source text.
2. Diagnose the biggest problems first: flat rhythm, hedging, vague wording, or excessive abstraction.
3. Rewrite in passes:
   - structure and paragraph flow
   - sentence rhythm and voice
   - word choice and specificity
4. Read the revised draft as if it will be spoken aloud. Fix any stiff or repetitive lines.
5. Call out tradeoffs if a stronger rewrite changes emphasis or register.

## Use This Skill Instead Of `humanizer` When

- The draft is long or high stakes
- The user wants a stronger voice
- The user explicitly mentions tone, rhythm, style, clarity, or flow
- The draft needs a meaningful rewrite, not just AI-tell removal
