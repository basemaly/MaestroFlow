# Humanizer

Use this skill for a fast cleanup pass when text feels canned, overly polished, or obviously AI-written.

## Focus Areas

- Remove assistant-style filler, hype, and generic enthusiasm
- Replace vague abstractions with direct, concrete wording
- Break repetitive sentence patterns
- Preserve facts and the user's intent

## Workflow

1. Read the full text before editing.
2. Remove conversational scaffolding like "great question", "I hope this helps", and generic wrap-up lines.
3. Replace inflated or promotional wording with plain, specific language.
4. Vary sentence rhythm so the draft does not read like evenly sized model output.
5. Do one final pass looking only for leftover AI tells.

## Output

- Briefly summarize what changed.
- Return the revised text.
- Note any places where stronger source material or user voice would still help.
